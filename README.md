# project-4-school-management-application
<p>This School Management System is an all-in-one application designed to streamline administrative tasks in an educational setting. Key features include registering student and teacher registration, detailed profile management, an attendance tracking system, and an efficient grading module to track academic progress.</p> 
<hr>
The code is written by me using python and the UI is generated with the help of gemini.
To run this code use the command streamlit run app.py . If streamlit is not installed in your code editor, run the command pip install streamlit on the terminal. and <b><i>also run pip install streamlit pandas</i></b>
